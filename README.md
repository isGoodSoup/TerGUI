## TerGUI
Un juego de Tres en Raya (Tic-Tac-Toe) muy simple, reescrito desde cero en Java 21 usando Swing para la interfaz gráfica.

## Qué es?
TerGUI es una implementación básica del clásico juego de tres en raya — donde dos jugadores colocan sus símbolos (X y O) en un tablero de 3×3. El primero que consigue tres en línea — ya sea horizontal, vertical o diagonalmente — gana. Si se llenan todas las casillas sin ninguna línea, el juego termina en empate. Tal y como dije, como el juego clásico.

Este proyecto está pensado como una versión sencilla para aprender Java y Swing GUI, sin lógica de IA avanzada, sólo interacción básica entre dos jugadores.
